# Smart Canteen Application

A modern, fully-functional canteen management system with user authentication, food management, and order processing.

## 🚀 Getting Started

To run the application, you need to start both the backend server and the frontend client.

### 1. **Backend Server**
```bash
cd server
npm install
npm run dev
```
The server will run on `http://localhost:5000`.

### 2. **Frontend Client**
```bash
cd client
npm install
npm start
```
The client will run on **`http://localhost:3000`**.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), Axios, Lucide-React
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Auth**: JWT
